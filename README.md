# Pokedex Inacap

Este es un proyecto de Pokedex hecho con Next.js y Tailwind CSS, es una recreación de un
proyecto que hice para la asignatura de Desarrollo de Software en 2024 con React.

## Características

- Buscador de Pokémon por nombre
- Lista de Pokémon
- Detalles de cada Pokémon

## Installation

Use [pnpm](https://pnpm.io/) para instalar las dependencias.

```bash
pnpm install
```

## Como usarlo

```bash
pnpm run dev
```
